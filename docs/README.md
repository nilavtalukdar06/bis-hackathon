# Project docs

This folder contains concise guides extracted from `project.txt` to help getting started, configuring auth, TRPC/rate-limiting, and deploying the boilerplate.

## Files

- `quick-start.md` — minimal local setup and commands
- `auth.md` — BetterAuth server/client setup and Google OAuth notes
- `trpc-rate-limit.md` — how TRPC is wired and how to adjust rate limits
- `deploy.md` — production checklist & common deployment notes

See `project.txt` at the repo root for the full single-file documentation.
